# Telegram-бот написанный на Python с использованием aiogram 3 и с БД SQLite

## Телеграм-бот для витрины и услуг, а также сбора заявок. Может подойти для малого бизнеса: консультации, продажи и формы обратной связи.

## Возможности

- FSM-воронки(опросы, анкеты да/нет сценарии)
- Inline и Reply кнопки
- Сбор заявок (Имя и телефон)
- Уведомление администратору в Телеграм
- Сохранение заявок в Базе Данных SQLite
- Автонапоминание пользователям(таймеры/автопуши)
- Логирование действий пользователя

## Зависимости
Python 3.10+
aiogram 3
SQLite 
asyncio
FSM

## Структура проекта
bot.py - основной файл самого бота
db.py - работа Базы Данных
keyboards.py - Inline и Reply кнопки
states.py - FSM-состояния
.env.example - пример переменных окружения

## Запуск
1. Установка зависимостей: pip install -r requirements.txt
2. Создать у себя .env по примеру .env.example:
   BOT_TOKEN - ваш токен от Телеграм-бота(можно узнать через @BotFather)
   ADMIN_ID - ваш айди аккаунта в телеграм(важно, не юзернейм или никнейм, a уникальный числовой айди)

3. Запустить:
   python bot.py
